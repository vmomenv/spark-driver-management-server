`pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple`
FastAPI: 用于构建Web API。
`pip install fastapi`

uvicorn: 用于运行FastAPI应用程序。
`pip install uvicorn`

pydantic: 用于数据验证和模型。
`pip install pydantic`

passlib: 用于密码哈希。
`pip install passlib`

jose: 用于JWT（JSON Web Token）支持。
`pip install python-jose`

`pip install python-multipart`
